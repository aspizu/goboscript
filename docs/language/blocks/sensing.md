# Sensing Blocks

### ask () and wait

```goboscript
ask question;
```

```scratchblocks
ask (question) and wait
```

### set drag mode [draggable]

```goboscript
set_drag_mode_draggable;
```

```scratchblocks
set drag mode [draggable v]
```

### set drag mode [not draggable]

```goboscript
set_drag_mode_not_draggable;
```

```scratchblocks
set drag mode [not draggable v]
```

### reset timer

```goboscript
reset_timer;
```

```scratchblocks
reset timer
```
