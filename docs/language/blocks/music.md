# Music Blocks

### play drum () for () beats

```goboscript
play_drum drum, beats;
```

```scratchblocks
play drum (1 v) for (beats) beats
```

### play note () for () beats

```goboscript
play_note note, beats;
```

```scratchblocks
play note (note) for (beats) beats
```

### set instrument to ()

```goboscript
set_instrument instrument;
```

```scratchblocks
set instrument to (instrument)
```

### rest for () beats

```goboscript
rest beats;
```

```scratchblocks
rest for (beats) beats
```

### set tempo to ()

```goboscript
set_tempo tempo_;
```

```scratchblocks
set tempo to (tempo_)
```

### change tempo by ()

```goboscript
change_tempo tempo_;
```

```scratchblocks
change tempo by (tempo_)
```
