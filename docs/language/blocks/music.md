# Music Blocks

### rest for () beats

```goboscript
rest beats;
```

```scratchblocks
rest for (beats) beats
```

### set tempo to ()

```goboscript
set_tempo tempo_;
```

```scratchblocks
set tempo to (tempo_)
```

### change tempo by ()

```goboscript
change_tempo tempo_;
```

```scratchblocks
change tempo by (tempo_)
```
