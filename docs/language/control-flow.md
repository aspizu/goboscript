# Control Flow

## repeat n times

```goboscript
repeat n {
    # code
}
```

![](../assets/repeat.png){width="100"}

## repeat until condition

```goboscript
until condition == true {
    # code
}
```

![](../assets/until.png){width="200"}

## for each loop

```goboscript
for x in n {
    # code
}
```

![](../assets/foreach.png){width="200"}

## for loop

```goboscript
for x = 0; x > n; x++ {
    # code
}
```

![](../assets/for.png){width="100"}

## forever loop

```goboscript
forever {
    # code
}
```

![](../assets/forever.png){width="100"}
